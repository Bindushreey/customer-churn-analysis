# Customer Churn Analysis (Python)

This project performs exploratory data analysis (EDA) to understand customer behavior and identify patterns influencing customer churn.

## Files
- eda.py : Python script used for data cleaning, transformation, and analysis

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib

## Key Tasks Performed
- Data cleaning and preprocessing
- Exploratory data analysis
- Pattern identification
- Visualization of churn drivers

This project demonstrates hands-on experience in analyzing structured datasets and extracting meaningful insights using Python.

Outcome:
Generated insights to understand customer attrition patterns and improve retention strategies.
